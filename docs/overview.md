# プロジェクト概要

## Rules

- この文書では目的、MVP 範囲、大方針だけを決める
- 実装詳細や技術詳細は `docs/technical-design.md` と `docs/architecture.md` に分ける
- 目的や MVP の境界を変えたら `docs/requirements.md` を含む関連文書も更新する

## 目的

このプロジェクトは、単一の画像を AI で高品質に 2 倍、3 倍、4 倍へ拡大する Windows 向け GUI アプリを作るためのものです。

狙う体験は次のとおりです。

- エクスプローラーから画像を開ける
- ウィンドウへドラッグアンドドロップできる
- 入力画像をプレビューできる
- GPU を使って AI アップスケールを実行できる
- 出力画像を保存できる

初期段階では配布を目的にしません。ただし、あとで配布対応をしやすいように、構造と UI は最初から整理して実装します。

## 現在の MVP 範囲

### 対象

- Windows デスクトップアプリ
- NVIDIA GeForce GPU 前提
- 画像 1 枚のみ処理
- 2 倍、3 倍、4 倍拡大に対応
- GUI 操作前提

### 入力方法

- ファイル選択ダイアログ
- ドラッグアンドドロップ
- 起動引数で渡されたファイルパス

### MVP に含めないもの

- 複数画像の一括処理
- 動画対応
- 学習機能
- インストーラー作成
- 配布自動化
- CPU を主対象にした実装

## 現時点の方針

- 言語は Python を第一候補とする
- GUI は PySide6 を使う
- 推論は PyTorch + CUDA を使う
- 構成は `ui / domain / infrastructure` を基本にする
- ユースケースは `domain/usecase` に置く
- 既存の学習済みモデルを使い、自前学習はしない
- 単一画像処理の内部構造を、将来の複数画像処理へ拡張しやすい形にする
