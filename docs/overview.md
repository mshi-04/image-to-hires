# プロジェクト概要

## 1. 目的

このプロジェクトは、画像を AI で高品質に 2 倍、3 倍、4 倍へ拡大し、複数ファイルをキュー方式で順次処理できる Windows 向け GUI アプリを作るためのものです。

狙う体験は次のとおりです。

- エクスプローラーから画像を開ける
- 選択したファイル名一覧とキュー状態を確認できる
- デノイズと拡大率を指定できる
- GPU を使って AI アップスケールを実行できる
- 出力画像を入力と同じ拡張子で自動保存できる

初期段階では配布を目的にしません。ただし、あとで配布対応をしやすいように、構造と UI は最初から整理して実装します。

## 2. 現在の MVP 範囲

### 2.1 対象

- Windows デスクトップアプリ
- NVIDIA GeForce GPU 前提
- 画像を 1 枚または複数枚処理
- 複数ファイルはキュー方式で順次処理
- 2 倍、3 倍、4 倍拡大に対応
- GUI 操作前提

### 2.2 入力方法

- ファイル選択ダイアログ

### 2.3 MVP に含めないもの

- 複数画像の並列処理
- 動画対応
- 学習機能
- インストーラー作成
- 配布自動化
- CPU を主対象にした実装

## 3. 現時点の方針

- 言語は Python を第一候補とする
- GUI は PySide6 を使う
- 推論は PyTorch + CUDA を使う
- 構成は `ui / domain / infrastructure` を基本にする
- ユースケースは `domain/usecase` に置く
- 既存の学習済みモデルを使い、自前学習はしない
- 1 ジョブ単位の処理を積み上げて、複数ファイルを順次処理できる形にする
